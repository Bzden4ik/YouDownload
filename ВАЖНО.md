# ⚠ ВАЖНО — YouDownload

## Первый запуск

```
cd C:\Users\warfa\Documents\GitHub\YouDownload
npm install
npm run dev
```

## При первом запуске
Приложение само скачает `yt-dlp.exe` (~10 MB) с GitHub в папку `bin/`.  
Нужен интернет. После этого работает полностью офлайн.

## Требования
- **Node.js 18+** — https://nodejs.org
- **ffmpeg** (рекомендуется для слияния видео+аудио в 1080p+)  
  Скачать: https://ffmpeg.org/download.html → добавить в PATH

## Сборка .exe
```
npm run package
```
Результат в папке `dist/`.
